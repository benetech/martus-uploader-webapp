How it works?

You are an existing Martus user and want to submit your records to Martus server. However, your internet connectivity is poor or you don't want to connect to Martus server from your home computer ?


With Martus Uploader, you can upload your encrypted Martus records to the Martus server from any PC. Export your records as "Uploadable Zip" file and drop the zip file on this website. Wait for the encrypted confirmation text. You can import the text file in Martus and confirm that the records were uploaded
